About

We Provide a system for an employee to request office assets with a flexible system so that it can increase employee's work productivity

Features Available in Asset Management System:

  • Dashboard
  • User's Profile
  • Request Item
  • Request History
  • Request Approval
  • Change Request Status
  • Add New Item
  • Register New User
  • Edit Profile
  • Update and Reset Password

What Asset Management System can do?

  • Profile page to see detailed information about yourself and edit your profile.
  • Access the dashboard to view all notifications about your asset request.
  • You can access your own portal for request an office asset
  • Helps you to see the status of recent requests and provides histories of your asset requests.
  • Admin could register new employee to request an office asset.
  • Your asset request will be previewed first by manager and manager will be decided to approve or reject your request.
  • You can change your password for the sake of security system.
  • And for you who forgot your current password, don't worry! We provide a system for you to reset your password.
